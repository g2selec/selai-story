The Challenge

Teaching AI a language
no AI has ever spoken.

SELPRO+ uses a proprietary DSL — a programming language for ladder logic that exists nowhere on the internet.

Every public language model — GPT, Gemini, Claude, Qwen — has never seen a single line of SELPRO DSL. There are no Stack Overflow posts, no GitHub repositories, no documentation indexed by any search engine.

When we set out to build an AI that could generate valid SELPRO ladder logic from plain English, we had exactly zero training examples to start with.

So we built the data ourselves. From first principles. Generator by generator. Until we had enough to teach a 3-billion parameter model to write code it had never seen a human write.

Problem 1

No real-world training data exists. SELPRO DSL is proprietary — not indexed, not public, not in any pre-training corpus.

Problem 2

The DSL has strict grammar rules — ANTLR4 parser, specific pin signatures, annotation syntax. Wrong output breaks the compiler.

Problem 3

Engineers phrase the same requirement in dozens of ways. The model must understand intent, not just keywords.

Our Solution

Synthetic data generation — 36 specialized generators that produce verified (prompt, DSL) pairs at scale, with zero parser failures.

Why not GPT-4?
Cloud APIs send your ladder logic to external servers. For an industrial controls company, that means proprietary machine configurations, production parameters, and IP leaving your network. SELAI runs entirely on your local server — no data leaves, no subscription cost, no internet required during operation.
⚡
Why LoRA fine-tuning?
LoRA updates only 0.1% of model parameters — enough to teach the model a new language (SELPRO DSL) without forgetting its general coding ability. Training 3,056 samples takes under 45 minutes on a free Kaggle GPU. Full fine-tuning would require 80GB+ VRAM and days of compute.
🔒
Why synthetic data?
Real PLC programs contain proprietary machine configurations that engineers wouldn't want in a training dataset. Synthetic data lets us generate thousands of clean, verified, diverse examples that cover every edge case — without touching a single production system.
